# CNCTD OS — Notion + n8n

Версионируемые n8n workflow для внутренней операционной системы CNCTD.

## Готово

- `workflows/01-weekly-dispatcher.json` — сканирование 9 отделов, расчёт нагрузки, балансировка недели и обновление Notion Dispatcher.
- `workflows/02-dispatcher-event-journal.json` — ежедневный неизменяемый снимок диспетчера в журнал событий.
- `workflows/03-weekly-analytics.json` — недельная аналитика и сравнение с предыдущей неделей на основе журнала.
- `docs/INSTALL-RU.md` — пошаговая ручная установка в n8n.

## Архитектурный принцип

Notion хранит задачи и является источником истины. n8n только читает, рассчитывает и обновляет агрегаты. Недельный диспетчер никогда не содержит копий задач.

## Быстрый старт

См. [инструкцию установки](docs/INSTALL-RU.md).

## Безопасность

Credentials и токены не хранятся в репозитории. После импорта credential назначается вручную в интерфейсе n8n.
